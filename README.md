# Siddharth Bhamare — Full Stack Developer Portfolio

A high-end, cinematic, and responsive personal portfolio built with React and Framer Motion. This project features a minimalist dark-themed aesthetic inspired by modern developer-centric designs, characterized by bold typography, neon green accents, and fluid motion.

![Tech Stack](https://img.shields.io/badge/Built%20With-React%20%2B%20Vite-blue?style=for-the-badge)

## 🚀 Experience the Journey

The portfolio begins with a **cinematic name reveal** ("SIDDHARTH") followed by a **neon green curtain transition** that unveils a sleek, interactive workspace.

### Key Features
- **✨ Immersive Preloader**: Custom letter-by-letter SVG/motion animation with a curtain reveal.
- **🌌 Interactive Background**: Smooth, canvas-based floating particle system that responds to the environment.
- **🖱️ Custom Cursor**: A sophisticated dual-element cursor (dot + lerped ring) that reacts to interactive elements.
- **📱 Responsive Sections**:
  - **Hero**: High-impact headlines with real-time stats.
  - **About**: A clean, typography-focused introduction.
  - **My Stack**: Categorized skills (Languages, Frontend, Backend, Tools) with visual badges.
  - **Selected Works**: Interactive project list with grayscale-to-color hover effects and direct GitHub links.
  - **Contact**: Minimalist footer with quick-access social and email links.
- **🧭 Navigation**: Slide-in drawer menu with color-coded navigation dots and smooth internal scrolling.

## 🛠️ Built With

- **[React](https://reactjs.org/)** - Core frontend library.
- **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling for fast builds.
- **[Framer Motion](https://www.framer.com/motion/)** - Production-ready motion library for React animations.
- **[Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)** - Custom design system using modern tokens and flex/grid layouts.
- **[Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)** - Performance-optimized particle animations.

## 🏗️ Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/just-sid04/Portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
